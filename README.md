# Awwa Stories - Personalized Storybook Generator (Phase 2)

AI-powered personalized storybook for **Awwa Stories** (UK).

## Features
- Upload child's photo → AI generates personalized anime character
- Supports **girl** and **boy** versions
- 5 scenes personalized with child's face
- PDF download

## Tech Stack
- PuLID (anime face generation)
- Claude API (feature extraction)
- fal.ai / flux-pro/kontext (face replacement)
- Streamlit (UI)
